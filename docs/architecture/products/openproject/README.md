# OpenProject Security Architecture

OpenProject is a conventional web product compared with OpenClaw, but it still
needs a product security view because it carries persistent data, admin roles,
and shared platform dependencies.

## Scope

This product view should remain focused on OpenProject-specific security
concerns. Shared platform services stay documented under platform and component
views.
