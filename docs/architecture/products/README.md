# Product Security Views

Product views describe security architecture, threat models, and required
controls that are specific to one product.

## Current Products

- [`openclaw/README.md`](openclaw/README.md)
- [`openproject/README.md`](openproject/README.md)

Use a product view when the product has a distinct user surface, privileged
execution path, or control profile that should not be hidden inside generic
platform docs.
