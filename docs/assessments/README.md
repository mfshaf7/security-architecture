# Assessments

Assessments are point-in-time security evaluations with durable evidence value.

## Structure

- [`platform/`](platform)
  - whole-platform assessments and executive summaries
- [`components/`](components)
  - component-specific assessments
- [`products/`](products)
  - product-specific assessments

Use `reviews/` for lighter-weight or change-driven review outputs. Use
`assessments/` for deeper assessment artifacts with broader evidence value.
